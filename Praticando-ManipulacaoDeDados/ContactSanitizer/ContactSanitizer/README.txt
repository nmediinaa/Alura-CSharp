# 🧹 ContactSanitizer

Este é um projeto de **Higienização de Dados** desenvolvido em C# 
como desafio prático para consolidar conhecimentos em **Expressões Regulares (Regex)** e **Manipulação de Strings**.

O objetivo é simular um cenário real onde recebemos uma lista de "leads" 
com dados sujos/não formatados e precisamos transformá-los em objetos estruturados e validados.
